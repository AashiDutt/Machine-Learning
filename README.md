# Machine Learning

***This repositry contains the code for Machine Learning in Python 3.***

# Requirements

**1. Python 3.5 +**

**2. Numpy [+mkl for Windows]**

**3. Matplotlib**

```
pip3 install matplotlib
```

**4. Pandas**

```
pip3 install pandas
```

**5. Scikit-Learn**

```
pip3 install scikit-learn
```

**6. Jupyter Notebook / Jupyter Lab**

```
pip3 install jupyter notebook
```
